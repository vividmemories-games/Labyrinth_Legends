# MVP Scope

## In scope

- [x] Home screen with navigation
- [x] World and level select
- [x] Gameplay screen with maze board
- [x] Path drawing (undo, erase, hint, go)
- [x] Fog of war around player/path
- [x] Level complete with stars
- [x] Shop (3 tabs, mock purchases)
- [x] Settings (audio toggles)
- [x] Daily challenge (date-seeded procedural maze)
- [x] Local progress save
- [x] Handcrafted JSON levels 1–20 in `assets/levels/world_1/`
- [x] Ancient Tech visual theme (gold primary, cyan paths)

## Deferred from original MVP spec (Phase 2)

- Basic trap cells (spike, pit, arrow) — keys, fog, relics cover Phase 1 tutorial

## Post-MVP

- Real AdMob integration
- Real IAP (RevenueCat or native)
- Firebase Analytics / Remote Config
- JSON level files in assets
- Procedural maze generation (daily challenge only in Phase 1)
- Hint path preview animation

## Explicitly out of scope (MVP)

- Multiplayer
- Online leaderboards
- Account system
- Energy/lives system
- Battle pass / cosmetics store
