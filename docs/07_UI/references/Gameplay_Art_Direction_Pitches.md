## Purpose

This reference captures broader gameplay-screen visual intent across multiple art-direction directions:

- monumental board readability
- calm strategic presentation
- relationship between environment, route lines, and explorer
- support-chrome restraint relative to the board

## Architectural Reading

Use this reference to validate:

- ancient / mystical gameplay tone
- board readability under different environmental treatments
- how support controls stay peripheral to the play space
- whether the gameplay screen still feels like a place rather than a control panel

Do **not** treat this reference as authoritative over architecture documents. It illustrates intent only.
