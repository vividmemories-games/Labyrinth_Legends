**Why:** WS0–WS7 defined philosophy through typography — but no document owned how icons, glyphs, seals, and markers communicate meaning across HUD, rewards, hazards, and navigation.

**Problem solved:** Prevents glyph-only controls, stock icon packs, casino reward seals, hazard/collectible metaphor drift, and unreadable micro-inscriptions on mobile HUD.

**Approach:** WS8 workshop extending Vision → WS0 → WS1 → WS2 → WS3 → WS4 → WS5 → WS6 → WS7; precedes `Icons.md` and icon token specs.
