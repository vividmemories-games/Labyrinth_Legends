## Task Summary

Refined global button label typography after weathered plate styling landed. Split `LLTextStyle.button` into **`buttonPrimary`** (17px Exo 2 w700, wider tracking for short CTAs) and **`buttonSecondary`** (14px w600, tighter tracking for long labels). Added **`LLShadow.buttonPlate`** for legibility on textured backgrounds. `LLButton` applies variant-specific styles app-wide.

**Out of scope:** New font families, Cinzel on buttons, per-screen label overrides.

## Motivation

**Why:** Human feedback — PLAY / Daily Challenge / Shop labels did not read well on weathered button plates (too light, awkward spacing).

**Problem solved:** Single `button` token with stacked letter-spacing override looked thin and stretched on home CTAs and long secondary labels.

**Approach:** Variant-specific functional typography per LLDL (Exo 2 only) + embossed shadow token; single change in `LLButton` propagates everywhere.
