## Task Summary

Human feedback after `0076`: the PLAY button looked worse without the weathered primary texture — the prior gold gradient + bronze plate styling was correct; only the empty notched corners needed fixing. Restored `button_primary_weathered.png` on the primary variant, kept the `ClipRRect` + gradient underlay from `0076`, and added a slight center scale (`textureScale: 1.14`) so ornate frame cutouts clip outside the rounded plate. No changes to gradient colors, borders, typography, or secondary/ghost/danger variants.

**Out of scope:** New flat primary texture asset, `LLFloatingButton` changes, Home layout tweaks.

## Motivation

**Why:** `0076` removed the primary texture to eliminate corner gaps; Human rejected that trade — styling regressed while corners were the only complaint.

**Problem solved:** Primary CTAs regain weathered bronze plate fidelity with clean rounded corners.

**Approach:** Keep structural plate fix (`ClipRRect` stack + gradient fill); restore texture; crop corner notches via center scale rather than removing the asset or changing LLDL tokens.
