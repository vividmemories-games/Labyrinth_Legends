## Task Summary

Replaced the broken wall-cell + combined autotile visual model with a full **edge-blocked maze engine** (schema v2) and **layered authored PNGs** (`tile_floor` base + `edge_*` overlays). Delivered: `CellEdges`/`canTraverse()`, v1→v2 migration, edge-carving `MazeGenerator`, `CellEdgeMask` rendering, 13 cohesive environment PNGs, `level_001`–`level_010` edge-test levels, and test/doc updates. Out of scope: procedural tile painters, combined autotile revival, levels 011+ progression unlock.

## Motivation

Adjacent `wall_fill` cells rendered as separate boxed frames; combined autotiles (`tile_n`, `tile_ne`, …) baked different floor textures per cell. Root cause: impassable **wall cells** in the engine while visuals tried to patch with autotiles. Edge-blockage on walkable cells + one floor base + transparent edge overlays matches design intent and fixes seams.
