## Task Summary

Replaced the straight vertical level list with a scrollable **winding world map** matching the Level Select mockup: `level_select.png` hero background, S-curve node anchors along the ruin path, curved dashed cyan connectors, gold/cyan level nodes with stars above, current level emphasis, world header, and star-progress footer. Level 1 starts at the bottom; map auto-scrolls to the current incomplete level.

**Out of scope:** Worlds screen (`/worlds`) card list redesign, world reward chest logic, per-level title labels on map.

## Motivation

**Why:** Human feedback — level map was a straight line; mockup shows a winding path on the citadel plate.

**Problem solved:** Level Select reads as an adventure path through ruins, not a generic list.

**Approach:** Normalized anchor layout + `CustomPaint` bezier connectors + positioned nodes over `LLScreenBackground`.
