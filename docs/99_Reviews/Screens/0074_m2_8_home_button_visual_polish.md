## Task Summary

Delivered MVP Home screen visual alignment with the master mockup and a **global weathered button system** applied across the app. Work spanned: AI-generated portal background, Cinzel title positioned over the gate lintel, `heroScale`/`veilStrength` tuning, global oxidized-bronze button tokens with texture skins, horizontal CTA inset, taller button plates, and secondary texture fixes (`BoxFit.cover`, full-bleed stone asset). No gameplay engine changes.

**Out of scope:** `LLIconButton` weathered skins, 9-slice button assets, live currency persistence, leaderboard route.

## Motivation

**Why:** Human direction to match `get back to basics.png` / master mockup — portal atmosphere, title placement, rusting weathered buttons everywhere (pause, shop, home, etc.).

**Problem solved:** Home used missing/cropped assets and glossy gradient buttons; logo blocked portal; secondary buttons showed white PNG margins; background was over-zoomed-out; CTAs touched screen edges.

**Approach:** Token + single-component changes (`LLButton`, `LLScreenBackground`) so all screens inherit; AI assets for background and button plates; layout `Stack` for title positioning.
