## Task Summary

Aligned the Home screen with the Splash/Home panel from the MVP master mockup (`docs/assets/mockups/ui_board_master.png`). Cropped and shipped portal background + logo PNG assets, restructured layout (coins left / gems right, stacked PLAY / DAILY CHALLENGE / SHOP, bottom icon dock), and added `veilStrength` on `LLScreenBackground` so the portal atmosphere reads through.

**Out of scope:** Live currency persistence, leaderboard route, decorative gold screen frame, button PNG skins, comma-formatted currency values.

## Motivation

**Why:** Human requested the Home hub match the mockup background, fonts, and button stack from `get back to basics.png` / master board.

**Problem solved:** Home previously referenced a missing `ui_mockup.png` asset and used a generic text-title shell layout that diverged from the approved MVP visual target.

**Approach:** Crop environment plate + logo from the canonical mockup, overlay LLDL widgets (no baked UI duplication), lighten veil for portal glow.
